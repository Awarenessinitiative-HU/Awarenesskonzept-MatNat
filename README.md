# Awarenesskonzept der MatNat

Das [Awarenesskonzept](https://www.initiative-awareness.de/informieren/awarenesss) der Fachschaften der Mathematisch-Naturwissenschaftlichen Fakultät der Humboldt-Universität zu Berlin ([MatNat Fakultät](https://fakultaeten.hu-berlin.de/de/mnf)). Erarbeitet von den Studierenden der [Awarenessinitiative HU](https://github.com/Awarenessinitiative-HU). Das Awarenesskonzept besteht hauptsächlich aus einen Code of Conduct und einen Leitfaden für die Awarenessarbeit.

## Veröffentlichungen

Das Konzept wird iterativ erarbeitet und mit der Zeit immer weiter verbessern. Die aktuellste Veröffentlichung ist stehts [hier](https://github.com/Awarenessinitiative-HU/Awarenesskonzept-MatNat/releases) zu finden.

## Guide

Speziell für die Mitglieder der Awarenessinitiative aber auf für alle anderen Interessierten eklärt ein kurzer [guide](GUIDE.md), wie dieses Repository funktioniert und wie alles aufgebaut ist. Wer nur an dem Endprodukt interessiert ist, kann sich gänzlich auf die Veröffentlichungen stürzen :)
